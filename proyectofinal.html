Descripción del Proyecto

Desarrollar una aplicación funcional que resuelva un problema real utilizando programación.

Permite gestionar información mediante formularios, bases de datos y consultas.

Resultados

Se obtuvo una aplicación funcional aplicando buenas prácticas de programación.

HTML & CSS

Para la estructura y diseño visual de la aplicación.

Para la lógica del sistema y la interacción del usuario.

MySQL

Para almacenar y consultar datos.

GitHub

Para el control de versiones del proyecto.
