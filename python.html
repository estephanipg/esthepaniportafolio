Python y sus Conceptos Generales

Introducción al lenguaje de programación Python

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y fácil de aprender. Fue creado por Guido van Rossum y es utilizado en desarrollo web, inteligencia artificial, análisis de datos, automatización y más.

Características de Python

  • Sintaxis sencilla y legible.
  • Multiplataforma.
  • Gran comunidad de desarrolladores.
  • Compatible con programación orientada a objetos.
  • Amplia variedad de librerías.

Conceptos Generales

Variables

Las variables almacenan información. 

nombre = "Juan"
edad = 20

Tipos de Datos

  • int → números enteros
  • float → números decimales
  • str → cadenas de texto
  • bool → valores verdadero o falso

Condicionales

if edad >= 18:
    print("Mayor de edad")

for i in range(5):
