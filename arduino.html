Arduino y sus Componentes

Introducción al hardware, sensores y electrónica

¿Qué es Arduino?

Arduino es una plataforma de creación de electrónica de código abierto, basada en hardware y software libre, flexible y fácil de utilizar. Permite a creadores, desarrolladores y entusiastas construir proyectos interactivos que pueden detectar y controlar objetos en el mundo real.

Placas Principales

  • Arduino Uno: La placa más popular y recomendada para principiantes.
  • Arduino Nano: Una versión compacta, ideal para proyectos con poco espacio.
  • Arduino Mega: Diseñada para proyectos más complejos que requieren más pines de entrada/salida.

Sensores Comunes

Sensor Ultrasónico (HC-SR04)

Utiliza ondas sonoras para medir la distancia entre el sensor y un objeto.

Sensor de Temperatura y Humedad (DHT11 / DHT22)

Mide las condiciones climáticas del entorno y es muy usado en estaciones meteorológicas caseras.

Fotorresistencia (LDR)

Un componente que varía su resistencia dependiendo de la cantidad de luz que incide sobre él.

Sensor de Movimiento (PIR)

Detecta cambios en la radiación infrarroja, ideal para alarmas y sistemas que detectan la presencia de personas.

Actuadores y Otros Componentes

  • LEDs: Diodos emisores de luz para indicadores visuales.
  • Servomotores: Motores que permiten controlar con precisión la posición angular.
  • Módulo Relé (Relay): Un interruptor magnético que permite controlar altas tensiones (como focos de 110V/220V) desde el Arduino.
  • Protoboard: Placa de pruebas para conectar componentes sin necesidad de soldar.

Ejemplo de Código: Parpadear un LED

El clásico "Hola Mundo" en Arduino consiste en encender y apagar un LED conectado al pin 13.

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
